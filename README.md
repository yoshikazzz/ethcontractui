# ETH-Contact-UI

## 1. Development Environment

- node: 8.9.4
- npm:5.6.0
- typescript: 2.9.1

## 2. Install

```
yarn
```

## 3. Run

```
yarn start
```

## 4. Build and deploy

```
npm run build

// All files you need will be placed inside public folder.
// Bring the folder to your server side to deploy
```
